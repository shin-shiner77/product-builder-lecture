# Project Blueprint: 로또 번호 추첨 사이트

## Overview
행운의 로또 번호를 추첨하고 시각적으로 즐거운 경험을 제공하는 웹 애플리케이션입니다. 현대적인 UI/UX와 모바일 응답성을 갖추고 있으며, 사용자와의 소통을 위한 제휴 문의 및 댓글 기능을 제공합니다.

## Implementation Details
- **Frontend**: HTML5, CSS3 (Modern Baseline features), JavaScript (ES Modules)
- **Design**: 
  - 다크/라이트 모드 지원
  - 미니멀하고 세련된 카드 레이아웃
  - 로또 번호 추첨 시 애니메이션 효과
- Features:
  - 로또 번호 자동 생성 (1-45 사이의 고유한 6개 숫자)
  - Wow 버튼을 통한 시각적 효과 (색상 변경 등)
  - Formspree를 이용한 제휴 문의 폼
  - Disqus를 이용한 댓글 기능 (구현 완료)

## Current Task: (None) - All requested features implemented.
